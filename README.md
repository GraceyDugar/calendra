# 🗓️ Calendra — Life Logger

> *Log your life, one range at a time.*

Calendra is a beautiful, interactive wall calendar that lets you log and track meaningful periods of your life — trips, work sprints, health events, study sessions, and personal moments — all tied to visual date ranges.

![Calendra Preview](https://images.unsplash.com/photo-1462275646964-a0e3386b89fa?w=1200&q=80)

🔗 **Live Demo:** [mycalendra.vercel.app](https://mycalendra.vercel.app)

---

## ✨ Features

- 📅 **Wall Calendar UI** — A physical wall calendar aesthetic with a dynamic hero image that changes every month
- 🗓️ **Date Range Selection** — Click a start and end date with smooth hover previews and visual range highlighting
- 🏷️ **5 Life Categories** — Tag entries as Travel, Work, Health, Study, or Personal
- 😊 **Mood Tracker** — Attach a mood to every logged period (Happy, Excited, Neutral, Stressed, Sad)
- 📝 **Rich Notes** — Write notes tied to specific date ranges, not floating in a void
- 📖 **Life Log Timeline** — A beautiful slide-in drawer showing all your logged entries with category filters
- 🗑️ **Delete Entries** — Remove any log you no longer need
- 💾 **Persistent Storage** — All entries saved to localStorage, survive page refreshes
- 📱 **Fully Responsive** — Works beautifully on mobile and desktop

## 🚀 Run Locally

**Prerequisites:** Node.js v18 or above

```bash
# Clone the repo
git clone https://github.com/GraceyDugar/calendra.git

# Go into the project
cd calendra

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---
## 🎯 How to Use

1. **Browse months** using the ‹ › arrows on the hero image
2. **Click any day** to set your start date
3. **Click another day** to set your end date — the range highlights in pink
4. **Choose a category** — Travel, Work, Health, Study or Personal
5. **Pick your mood** for that period
6. **Write a note** about what happened
7. **Click Save Entry** to log it
8. **Click "My Life Log"** button to view all your entries in the timeline

---

## 💡 Why Calendra?

Your phone's notes app can't do this. Calendra ties notes to **visual time ranges** — so you can see *when* things happened, *how long* they lasted, and *how you felt* during that period. It's a personal life logging tool for people who think in timelines.
